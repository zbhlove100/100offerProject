中国狙击手训练教程
　　
　　一个良好的狙击手需要具备三个条件，优秀的射击技术只是所有狙击技巧中最后也是最基础的一点，纪律及细心才是最重要的元素，狙击手并不是拿着枪胡乱扫射的杀人狂，他们不必要时不会杀害无威胁非武装人员，例如妇女，儿童，牲畜，建筑房屋等，狙击手有时是一个人，也可能2、3人，这要看任务的性质而定，单为射击而盲目的向目标射击只会增加被发现、俘虏及杀害的机会，纪律和耐心是非常重要的，尤其是要面对射与不射的时候。在成为一个狙击手之前必须先问问自己：你是否很怕冷或怕热？是否很容易发怒？愤怒会使你不小心以及做一些不寻常的行为，这是最坏的一点。有没有尝试过一个人单独过一个星期甚至更长时间？能否没有朋友，没有家庭，没有通讯，没有联络或什么都没有？是否有过一个人露营？在一个地方内什么人也没有，你会觉得怎么样？你又会在那里做什么？狙击手并不一定是个孤独者，但事实上如果你每日的生活都不能缺少其他人的存在的话，狙击手一定不是你最好的选择。 
　　为什么这些很重要呢？一个狙击手只为了开一枪而爬行一整天，有时还可能什么也找不到，你是否能一枪不发而放弃任务？有时你见到目标而开枪的机会只有三秒钟，假如你在做白日梦，吃饭又或者其他无为事情，你便失败了。你必须了解你的任务、位置并等待目标出现，这是你需要队友的原因，一个人用望远镜连续观察超过20分钟是十分困难的，你必须整天保持不动以避免敌人的发现，这听来十分容易但其实十分困难，一个初学钓鱼者要将鱼线留在水面以下超过三分钟是不可能的，因为他们总想要把鱼钩提上来看看，如果你有打猎的经验你会知道在猎物面前保持不动是如何艰难，更何况现在的你面对的猎物是一看见你就会向你开枪的人呢？你怎样解释“细心”一词？对一名优秀的狙击手来说细心就是他的一切，而且影响着他的决定，单是细心已经可以令一个狙击手成功，狙击手行动前必须决定要身处哪里，怎么走，怎么去，带什么装备，用什么伪装，如何通讯，行动时如遇紧急情况应该如何，任务完成如何撤退，无法完成又怎样避免损失？一个狙击手必须由开始到结束详细思考所有程序，其武器及子弹补给装备配置才会产生效果，射击术是最后的要素，一个狙击手在任何情况下都需要在最远的有效射程射击目标，距离等于撤退的生命时间。如果要做到射法纯熟，最少需要练习15000-20000发子弹才算得上是合格练习。 
　　一、狙击手的历史 
　　早在17世纪初光学仪器发明的同时狙击手的发展即已经开始了，但在整整两个世纪中战术狙击技术的发展却因为枪械工艺发展缓慢而未曾崭露头角。直到美国独立战争期间美国义勇军的一位夏普少校以特别改造过的枪械与各种小技巧（例如将每发圆球形的子弹都以浸透油脂的鹿皮包住，在装弹时不仅更方便，射程与精度亦随之提升）带领一队独立、高机动性的枪手以当时来看是不可思议的长距离（大约300-350米）精确狙击射杀了多名英军高级军官，多次以及小代价挽回一场战役的局势，而从此英文就出现了一个新的单词――Sharp shooters（夏普的射手们），意指射击精确而又冷静沉着的射手。为了长时间的贴腮瞄准与防止野外环境导致分心，夏普射手们在进行狙击任务时经常戴类似今日特植慷哟鞯拿嬲郑虼擞钟辛艘桓鲂旅治狭arkman，直译为戴面具的人，在英文中也指枪法或射击准确的人，而这两个词在今日则由Sniper这个词所取代，也就是“狙击手”。美国南北战争期间Hiram Berdan将军曾招训一个特别小组，针对射击与狙击的精要一一加以阐述，此举可视为美军正式训练狙击手的发源。 
　　1、第一次世界大战时期（1914 - 1918） 
　　1914-1915年冬天，德国军队在狙击方面占据领先地位。光学工业的发展为其提供了基础。借助于98型毛瑟枪德军狙击手取得了血淋淋的收获。敌方的军官、观察员和士兵只要暴露在战壕外面几秒钟就会被击毙。相比之下协约国地反应却是很慢的：他们试验了各种光学材料、精密步枪、伪装服饰、取消和隐蔽了多种明显的标志以便重新取得对德军的优势。在一些军官的压力下英军开始兴办一些特殊学校培养狙击手、观察手和侦察兵。法军在挑选精密武器方面有很大困难，直到战争末期才解决这个问题。但是他的伪装服在整个战争过程中都是样板。美军的狙击情况可以通过在海军陆战队中流行的一句话来说明：“一名海军陆战队狙击队员加一支步枪是世界上最危险的敌人”。借助于他们的Springfield M1903型步枪，美国海军陆战队队员无需瞄准镜就可以击中400米内的目标。通常美军的射手都被派到英军的学校去学习，以便更好地了解未知的敌人，改进已经远远超过欧洲人的射击技术。 
　　2、第二次世界大战时期（1939 - 1945） 
　　纳粹德国军队装备着与其使命相适应的武器和光学设备以及其狙击学说和狙击手进入了这次战争。狙击手的训练、武器和服装的供应都受到极大重视并且得到了极大的改进。主要问题是精密武器已经不足以供应德国国防军和党卫队的一百多个师使用。那个时期的照片和文件说明德军经常不得不使用莫辛·；纳甘1891-30型和俄罗斯产SVT型口径为7.62毫米的半自动步枪或捷克生产的毛瑟步枪。而德国生产的枪械则不足，例如98K型毛瑟枪和G43型步枪，口径均为7.92毫米。 
　　德军较好的狙击手基本都在东线服务。例如第三师著名的玛蒂亚斯·；里则纳尔，他共枪杀了“经确认的”345名敌方军人。但是这个数字有夸大的成分，因为在德军中一名被击毙的死者要经过一名军官，一名副官或两名士兵证实。很有可能是他们重复统计了数字。在前线作战的纳粹党卫军中对狙击手特别关注，实行奖励制度，鼓励他们去枪杀更多的敌人。对于成绩突出的狙击手可以享受与纳粹高级领导人共度周末狩猎的殊荣。 
　　英军以不寻常的速度做出了反映。重新设立了狙击学校，实行新的招募制度，在每个营内设立一个八人狙击小组。在挪威战争、1939-1940年的法国战争以及北非战争中均采用了狙击手作战。由于预料到欧洲战争的爆发，从1943年起英军开始更好的训练狙击手，提供精良的武器和伪装服，支持高效的教程和条令，并用于战争进行验证。 
　　苏军拥有54160支1938年的莫辛·；纳甘1891-30型步枪，在狙击手训练方面也是位列前茅的。但是这也没有使它在1939-1940年冬季战争中免遭芬军重创。只是这使得苏军更深刻地认识到了狙击手在防御战中的作用。在芬兰的狙击手中第34轻装步兵团的Simo Hauml在不到五天的时间内打死五百多名苏联红军，他也是历来成绩最好的狙击手，在其国内享有盛名。1940年6月3日他被苏联士兵打伤了脸部。从战争开始苏联就拥有优良的步枪和瞄准镜以及不同的伪装服，如类似英国的“傻瓜”伪装服。它支持共青团的射击俱乐部，而红军则拥有令人生畏的射击手后备力量。利用莫辛·；纳甘1891-30型和俄罗斯产SVT型口径为7.62毫米x54r的半自动步枪，苏军使轴心国在苏联受到的伤亡数量占了总伤亡数的一半。据报道苏军著名的狙击手瓦西里·；扎伊采夫共打死敌军280多人，但是还有其他一些狙击手超过了他，例如女狙击手柳德米拉·；巴普利琴科，她打死了经过核实的309名敌人。但是瓦西里击毙了德国狙击学校的校长，号称“狙击王”的科宁斯。美军仍然使用Springfield M1903型步枪，上面带有放大12倍的Unertl瞄准镜。陆军则更喜欢使用装有放大2.5倍瞄准镜的新式Garand M1c半自动步枪，因其火力较强。海军陆战队模仿英国也开办了狙击手学校。在瓜达尔卡纳尔和塔拉瓦战役的初期美军更加理解了狙击手的作用，战役中频繁的使用了狙击手。 
　　3、朝鲜战争（1950 - 1953） 
　　在美国第二次世界大战末期标志着狙击手学校的解体。只是海军陆战队还保留有一支神射手培训队。汉城等战役后狙击手作用重新受到重视。他们当时使用的主要是Garand M1c半自动步枪。开始时狙击步枪都保存在副官管理的军火库中，只在需要时才发给优秀射手。后来冲突陷于长期困境中，对狙击行动的认识更趋于合理化。新规程更加开明，上面带有放大8倍的Unertl瞄准镜的Winchester 70 Target型步枪也发给了有资格的狙击手。一些狙击手甚至用上了12.7毫米口径的Browning M2重机枪，由于它可配有望远瞄准镜并且可单发点射，所以能够射击2000多米远。 
　　英军的狙击学校关闭了但是皇家海军除外，只此一次，在英国军中没有狙击手，这种情况持续到1968年。 
　　4、阿尔及利亚战争时期（1954 - 1962） 
　　这时期的狙击手使用的是MAS 49和49/56步枪，安装有日间瞄准镜或红外瞄准镜，但是狙击手的作用更像是一种定点支援人员而不是一个行动区的独立行动人员 
　　5、越南战争时期（1965 - 1975） 
　　初到越南时美国海军陆战队狙击手使用Garand M1c、M1d、Springfield M1903 A4、M1A和M14步枪。不坚固的Winchester 70被Remington 700 Varmint卡宾枪取代，后者口径为7.62毫米，上面安装了带有3-9x40变焦镜头的Redfield瞄准镜。有记录显示这种枪曾经击中过900多米处的目标。 
　　直到1990年初M24 SWS步枪问世为止美国陆军装备的一直是M21步枪（它是口径7.62毫米的M14步枪的改进型产品，更加精密）上面安装了带有3-9x40变焦镜头的可调视距的Redfield望远瞄准镜，镜头内含有可计算距离的十字线。可用于特种行动。 
　　陆军的一些师和海军陆战队开办了自己的狙击手学校，在一些人士的努力下狙击手的水平有所提高，如狙击手Chuck Mawhiney枪杀了103人，Carlos Hathcock 83人。 
　　越南人民军的狙击手以莫辛·；纳甘1891-30型和苏联提供稀少的SVD Dragunov狙击步枪与美军相对抗。 
　　6、黎巴嫩战争时期（1976 - 1990） 
　　据记载狙击手在此期间使用的武器有口径7.62毫米带瞄准镜的FAL，Garand M1、莫辛·；纳甘1891-30型和稀少的SVD Dragunov，口径5.56毫米有Colt瞄准镜的M16A1以及法军供给黎巴嫩正规军的少量口径7.5毫米装有APXL806的FRF1。派驻黎巴嫩的联合国临时部队在南黎巴嫩的那些分队以及在贝鲁特的FMSB部队都用狙击手对付各种民兵部队的骚扰和进攻。 
　　法国第三和第八海军陆战队伞兵团的伞兵从78年到达后开始用带有夜视瞄准镜的FRF1部枪还击提尔区巴勒斯坦突击队的进攻。1982年开始FMSB的部队全部装备了FRF1步枪。同时意大利的狙击手装备了德国HK G3 SG1。 
　　美军依靠侦查狙击手对付散兵游勇的射击和进攻。海军陆战队采用口径为12.7毫米的步枪（如Iver-Johnson和McMillan）来克服其口径为7.5毫米的M40A1步枪对远距离目标杀伤力低的缺点。 
　　1982年春天以色列狙击手基本都使用带有E1 Op Nimrod 6x40瞄准镜的美式M14步枪取代1973年交付使用的口径为7.62毫米有Wild型放大4倍瞄准镜的98K型毛瑟步枪。 
　　7、格林纳达战争时期（1983） 
　　在“盛怒”行动中美军第二、第三别动队和第82空降师的狙击手参加了战斗。他们使用的是M21步枪。海军陆战队员狙击手击毙敌人的记录是三人。这是越战后美军第一次正式的狙击纪录。 
　　8、巴拿马战争（1989） 
　　仍然使用M21步枪的美军第82空降师和从1987年装备了口径为7.62毫米的新式M24 SWS步枪的第七步兵师，以及装备M40A1和M16A2的海军陆战队的狙击手参加了战争（其中一名狙击手在这次战争中有击毙三人的纪录）。 
　　9、海湾战争时期（1990 - 1991） 
　　这里很适合使用步枪作战。美军陆军狙击手使用的是M24 SWS步枪，海军陆战队的侦察狙击兵使用的是口径为12.7毫米的Barrett M82A1（亦称SASR）。特种部队和海豹突击队使用的是口径为12.7毫米的McMillan步枪。曾有一著名的战例：两名海军陆战队员两枪就将伊拉克的一辆履带运输车打瘫痪了。美军狙击手成绩有所提高，最高个人记录是打死15人。英军狙击手使用L96A1和L42。一些老狙击手仍然使用Lee-Enfield步枪。法军狙击手使用FRF2带有APXL806瞄准镜以及少量的Barrett和HB308。 
　　10、巴尔干战争时期（1992 - 2001） 
　　在这场战争中很难将众多的武器和部队搞清楚，但是有一点是肯定的，这里是展示狙击装备现代化、大口径化和训练改进的舞台。战争开始后的十年中狙击领域发生了巨大变化。 
　　法军是巨变的样板。1992年它装备了FRF2型狙击步枪，上面装有APXL806型瞄准镜（第二外籍伞兵团第四连的狙击手装备的是供外籍军团士兵使用的Swarowski Habicht Nova瞄准镜和供海军陆战队员使用的Schmidt和Bender瞄准镜），特种部队装备各种口径为12.7毫米，上面装有Leupold瞄准镜的步枪。经多次试验后又选用了McMillan系列的M87、M87r和M93，后来又使用了受欢迎的口径为12.7毫米的PGM H é； cate II步枪。
