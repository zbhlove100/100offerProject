第七十一章 美国股市崩盘（下）
 　　来-看
　　1987年10月19日星期一，如同罗杰斯所预料的一样，美国股市这一天大崩盘，道·琼斯工业指数一天内跌了创记录的508点，跌幅高达%，而纳斯达克指数跌幅也21%。华尔街股市的暴跌，标志着美国股市持续了5年的牛市结束。就接下来一周时间里，美国股市仍旧持续爆跌，美国股票每天的平均跌幅都17%以上。
　　富有戏剧性的是，10月19日这天恰好是罗杰斯43岁的生日，美国股市这一天大崩盘，可谓是带给他一份丰厚的生日贺礼。
　　就这一周，凭借这罗杰斯的正确预见，以及购买期指下跌、卖空股票等多种而又恰当的操作，那100亿美元资金流获得了2，5倍的利润回报。其李韦德的那一部分，由20亿美元上涨到了70亿美元。
　　反观罗斯那边，罗斯所预言的要暴跌的日本股市却极为坚挺，日本股市1987年后3个月反而不断创历史高。由于错误预见股市崩盘从日本开始，因为罗斯日本股市上购买期指下跌、卖空股票，实际资金则投资到美国股市上购买美国公司股票等错误行为，结果罗斯成为华尔街股市暴跌大的亏损者。
　　罗斯的量子基金遭受了32%的财产损失，量子基金资产从26亿多美元下降到18亿多美元，罗斯和量子基金不到两周的时间内损失了亿美元，这其罗斯个人就损失了将近2亿美元。这也意味着，自从罗杰斯1981年离开量子基金以来。这6、7年罗斯和量子基金所有的辛苦努力，全部付之东流。这也是罗斯人生当的一次重大挫折。
　　对于罗斯的惨痛失败，作为老朋友的罗杰斯也不知道该说什么好了，并没有幸灾乐祸的嘲笑罗斯不停他的，而只是打电话慰问劝解了失意的罗斯。
　　罗斯是失败的，罗杰斯和李韦德却是成功的。罗杰斯且不去提。这次的美国股市崩盘，成功把20亿美元的资金变为70亿美元之后，李韦德只提走了其原来的20亿美元现金，重存到他的瑞士银行匿名帐户上，而剩下的50亿美元则趁着美国公司股票的暴跌，大肆购买起美国各大军火公司的股票来。
　　由于美国股市的崩盘，美国各大公司股票的暴跌，很多美国各大军火公司，长期持有大部分股份的各大财团、各大家族、各大基金，也开始向外抛售部分股票，再加上原来日常股市上所流通的那部分股份，留给李韦德购买的空间自然大大放开。
　　1987年美国股市这场风暴过去之后，李韦德终便拥有了：洛克希德公司12%的股份，成为格罗斯家族、罗杰斯家族、加利福尼亚财团之后，洛克希德公司的第4大股东；马丁·玛丽埃塔公司16%的股份。成为洛克菲勒财团、梅隆财团之后，马丁·玛丽埃塔公司的第3大股东；诺斯罗普公司14%的股份，成为洛克菲勒财团、诺斯罗普家族、加利福尼亚财团之后，诺斯罗普公司的第4大股东；格鲁曼公司18%的股份，成为摩根财团之后，格鲁曼公司的第2大股东；雷神公司15%的股份，成为雷神公司的第3大股东；通用动力公司8%的股份，成为摩根财团、杜邦财团和另2家基金之后，通用动力公司的第5大股东；波音公司7%的股份。成为第一花旗银行财团和另3家基金之后，通用动力公司的第5大股东……
　　对比一下1986年年初国际某著名调查机构，所调查统计出来的美国?大军火公司排名：麦道公司、洛克希德公司、波音公司、诺斯罗普公司、马丁·玛丽埃塔公司、通用动力公司、格鲁曼公司、雷神公司、汤普森·拉莫·伍尔德里奇公司、联合技术公司。
　　可以现，除了排第一位的军火公司麦道公司，以及排前十名末尾的两家军火公司之外，排名二到八位的军火公司，李韦德都购买了一定数量的股票。没有购买麦道公司的股票，是因为麦道公司自从八十年代以来公司业绩一直不景气，股票不断下滑负债越来越多，对此李韦德感觉麦道公司的股票没有投资价值，而两位两家公司则是李韦德所不感兴趣的。
　　而李韦德购买这几家公司的这些股票，前后总共只花费了50亿美元，这平时简直是不可想象的。美国股市没有崩盘之前，李韦德只花这些钱是不可能买到这些数量的股票的，而且即使金钱足够也无法买到这么多的股份份额。而目前购买的这么多股份，也是李韦德所能承受的并且能够购买到的大量了。
　　由于李韦德是以埃里克投资公司购买这些股票的，所以这也引来了美国联邦政府和军方，以及各大财团的注意。当政府和军方确定这就是去年购买洛克希德公司股票，公司所有人是李韦德这个他们所熟悉的人之后便没有多管，毕竟李韦德作为美国公民的购买是合法的，当然对于埃里克投资公司的监控力加强是少不了的。而各大财团方面，因为李韦德邀请罗杰斯担任埃里克投资公司的高级顾问。这些大财团也理所当然的以为是罗杰斯家族的行为，所以他们也没有多大的反应。
　　当然随后，李韦德便接到了各大财团以各种形式的邀请和会面，通过罗杰斯当的介绍，李韦德和洛克菲勒财团、摩根财团、第一花旗银行财团、杜邦财团、加利福尼亚财团、梅隆财团，以及一些利益团体和家族进行了广泛的接触。这也让李韦德美国上层社会的影响力和关系网大大增加，对他今后的军火生涯自然是极为有利的。
　　洛克菲勒财团、摩根财团、第一花旗银行财团、杜邦财团、加利福尼亚财团、梅隆财团这个财团，加上波士顿财团、得克萨斯财团、克利夫兰财团、芝加哥财团这四个财团，构成了美国十大财团。可以说美国各行各业甚至包括美国联邦政府，基本上都有着这十个财团的身影，其影响力自然是绝对不容忽视的。要知道就是美国总统竞选，都少不了这十大财团背后的资金支持。
　　完成了对这些军火公司一定数量的股票收购之后，李韦德凭借着他手掌握的股份份额，和各大财团各大家族协调好之后，进入了这几家军火公司的董事会，成功的把触角伸到了美**火公司里，圆满完成了他一个阶段的任务目标。
　　此事过后一个多月，李韦德便迎来了一个另他大喜的消息。1987年12月，埃及军方向美国洛克希德公司购买。以3架-130大力神军用运输机为主包括其它武器装备内的，总共价值3亿美元的采购合同。
　　这个消息极大的刺激了洛克希德公司股票的上涨，各投资者把目光和资金纷纷投向洛克希德公司，使得洛克希德公司股票由这次美国股市崩盘的低点每股30多美元，很快回升并比原来价格还要高的，达到每股美元的价格。洛克希德公司成为了这次美国股票崩盘之后，第一批股票大幅回升并上扬的公司之一。
　　其实身为洛克希德公司董事会常任董事、洛克希德公司第4大股东的李韦德知道，为了挽回洛克希德公司的股票，经过董事会授权洛克希德公司的董事长和总经理等管理高层。可是耍了一个小手段才从埃及军方那里获得这份采购合同的。
　　1987年12月，埃及政坛极有势力和背景的，埃及议会议员雷拉·爱·泰克拉，建立了泰克拉公司，成为洛克希德股份有限公司的拿薪水的“顾问”。当时，洛克希德公司急于使埃及空军向它购买3架-130大力神运输机以及其它武器装备，向泰克拉公司转移了价值100万美元的“咨询费”，这才有了这份采购合同，才使得洛克希德股票出现这样好的成绩。
　　好消息不仅来自洛克希德公司，这一个多月里，随着美国联邦政府采取种种措施，美国股市开始逐渐复苏。这些军火公司也自然逐渐开始恢复，虽然进程很缓慢但也很明显，当然了有个别公司这次的打击下并没有恢复元气，因为这个别公司本身就够衰败的，这次美国股市崩盘只是加速了他们消亡的时间而已，就比如麦道公司。
　　当然对于这样的公司，李韦德可是一家也没沾，虽然他主要目的是为了进入美国各大军火公司的掌权团体，但明显没有投资价值的公司他也是不会去多费劲的。
