第十四节 被活捉的阿龙
　　
　　阿龙他是头晕脑涨的，强烈的窒息感让他的脆弱的大脑遭受到了前所未有的挑战，他的神经不断的挤压，都快要临近绷断了的程度，这是阿龙他以前出来没有训练过的，这是对他最具备杀伤力攻击，这无穷同时强大的攻击方式几乎是瞬间解决掉了号称中国乃至是世界范围内最强悍的特种兵――怒龙特种兵的战斗力。
　　就在阿龙的意识快要陷入昏迷之中的生活，那个原本如同铁钳一般的大手此刻居然是松了开来，在死亡的边缘放过了年轻而且幼稚的阿龙，看起来背后的这个神秘的人让他感到了一丝丝的害怕。
　　当然了，这些东西并不是差点昏迷掉了的阿龙他所能够知道的，他就是非常奇怪的为什么这个叫做千叶鸣一的人，他的能力是让那么的强悍，简直是达到了不是人的地步，他是突然间对于他这个素不相识的人突下杀手，就在他快点死亡的时候，他又突然间放手了。
　　“咳……咳……咳……咳”
　　阿龙他不断的咳嗽，以摆脱这个对喉咙的强烈刺激感，他的眼睛也有不少的泪水流了出来，使得他此刻的视线也变的十分的模糊，这让他有一种危险的感觉，因为要把任何的状况时刻的放在自己的心头这是身为特种兵对自己的最起码的要求，这个强烈的耻辱感让他非常的难受，而且也是这样的耻辱感让他以最快的速度摆脱了这样一个状态，他强迫自己把这个含有泪水的眼睛睁开。
　　阿龙他通过这模糊的视线看隐隐约约的看到了处于震惊状态之中的千叶鸣一，尽管看的不是特别的清楚，可是他还是能够非常他眼神之中的强烈的震撼感。他的那双原本是镇静的双手居然是出现了颤抖，在对阿龙他而言是一个非常困惑的事情，他刚才是什么也没有听到，而且阿龙他改变不认识千叶鸣一，所以他就不会明白千叶鸣一他此时的心情了。
　　易天他不快不慢的走了过来，冷声冷语的说道：“好了！现在你应该离开这里，要不然有什么麻烦的事情发生我可不能确定了。”
　　不知道是不是易天
　　他那冷冰冰的话语刺激了千叶鸣一还是其他的原因，反正他是摆脱了那因为过于震撼而陷入的短时间的大脑空白，他用难以理解的眼神盯着易天他的身上，然后轻轻的扫了阿龙他一眼，他说道：“今天看着尊敬的大人的面子上我就不在这里捣乱了，但是我绝对不会放弃的，还有我替我的主人给大人带来他诚挚的问候，主人在我来的时候希望通过我给大人带上一句话。”
　　易天
　　他那原本是万年冰山的脸终于是动容了，他那浓浓的大眉狠狠的纠结在一起，他冷冰冰的说道：“他要你带什么话？”
　　千叶鸣一他骄傲的说道：“易天！原本你是没有资格知道的，可是到了现在，算是你的福气了，我的主人他说了，只要大人答应他的要求，他可以为大人放弃一切，甚至是包括圣子在内。”
　　“什么？你说什么？”
　　易天他的镇定是此刻化为乌有了，他激动的说道：“你说什么？他真的这样的说吗？”
　　千叶鸣一他笑道：“当然是这样，我一直为我的主人不值，他为什么会这样的做，可是现在我明白了，我也知道大人也在听我的话，所以我算是把话带到了，希望到时候大人能够给我主人一个明确的答案，那么我也不算是冤枉的走了这么一遭，对了，听说天狗他来了，所以我希望你们小心一点，而且我们还听说了，他们带了一件特别的实验品要在这里测试一下。大人，我就告辞了！”
　　阿龙他是听了一大段莫名其妙的话，也紧接着这个一直在他的目前唧唧呱呱的家伙就像是鬼一般的消失了，他的大脑好一阵子拒绝接受这样一个事实，他摇摇头对场上唯一一个可能解答他的疑惑的人看去，他谨慎的问道：“请问！你们是人还是……”
　　易天他是满脸的痛恨的神色，他恶狠狠的看着阿龙，仿佛想要一口吞掉他一样，而且他的眼眶也好象因为过度瞪，使得都快要裂开了一样，这再加上他那僵硬的脸，就好象是有一具僵尸想要咬他一样。
　　大胆的阿龙也不免被这样的举动给吓上一跳，他低声低语的说道：“请问！你有必要这样的看着我吗？难道我做了什么对不起你的事情吗？”
　　易天他大声骂道：“你真的是没有用，早知道就不让他们那些人教你了，你看看那原本多好的苗子都成为什么鬼样子了，你垃圾一点也无所谓，可是你居然要害的我们的大人受这样的侮辱，这就是你不能原谅的地方。”
　　阿龙他是越来越糊涂了，他结结巴巴的说道：“你是在说我吗？好象我们并不认识啊？”
　　易天他是依旧的大声的骂了起来，说道：“谁希望认识你这样的垃圾，我真的想不到为什么大人要为你这样的一个垃圾牺牲那么多的东西。不值得啊！”
　　阿龙他本就是一个男人，而且是血气方刚之年，起初的那些谨慎不过是常年的特种兵训练之中所培养出来的一点谨慎，他在这接二连三的被一个强大而且陌生的人骂了半天，他忍不住回了一句，他说道：“你这个怪物，真的是不食人间的烟火，你根本就不是人，你凭什么说我们这些普通的人，要知道我可是人类之中最精锐的特种兵，当然了，这并不包括你这样的非人在内。”
　　原本以为易天
　　他会以暴力回迎他的挑衅，毕竟他们那些强大的存在都有自己的尊严，对于任何挑战他们尊严的人，都不会有什么好下场，可是在阿龙他的意料之外，一直是以恶言恶语相加的易天
　　他居然没有任何的反应，阿龙他好奇的看了看易天他的眼睛，发现在他的眼睛里面充满了迷茫，这一发现真的让他对这些神秘莫测的人有一种很强的好奇心，他们到底是谁，居然能够凭借一己之力翻江倒海甚至是改变天空中的云彩这样近乎天方夜谈般的神话。可是却不知道为什么他们老是为一些希奇古怪的问题而陷入莫名其妙的沉思。
　　突然间，阿龙他变成了一只灵活的猴子一样想要闪到旁边去，因为他感觉到了有一个东西从他身边飘了过去，就仿佛是一阵清风经过一样，但是这却不是什么清风，因为阿龙从中感觉到了一点点危险，可是阿龙他也不是什么猴子，况且就算是一只灵活的猴子也不见得能够逃的过去。
　　这个清风经过了他的身边的时候，立刻就变成了一只闻到了肉的饿狼一样，紧紧的把他困了起来，甚至是连那流动的空气也变成了凝固的水泥了，阿龙他是连眼皮都不能眨动了，他的肩膀上好像有了万钧之力一样，不断的施加到脆弱的阿龙的身上。
　　――轰――
　　阿龙他好像一个坚强的英雄一样倒了下去，阿龙他的神智是彻底的陷入了昏迷，他没有发现在这莫名其妙的重压之下，他的双脚所站立的地方已经是出现了龟裂，在原本是一个普通的人不可能做的出来的事情。
　　“哎！你怎么还是无法忘记往事呢？把他带回来吧！你自己好好的考虑自己的事情吧？”
　　不知道过了多久，反正是阿龙他从疲惫的身心之中苏醒了过来，他没有立刻睁开眼睛，因为他想起来了他是晕倒了，现在装晕是他唯一可以反击的机会，这样才能够让那些对他心存不轨的人放松他们的神经，但是阿龙他失策了，因为他考虑的是平常的人，尽管面对那些其他特种兵的逼供他也能够继续装下去。
　　“醒了就醒了吧？不要再睡了，到时候看不到好戏就不要怪我了。”
　　一个冷冰冰的声音在阿龙他的耳边响了起来，这可把阿龙他吓了一跳，他以为是易天
　　那个混蛋，可是他觉得有一点不对劲，因为易天他的冷冰冰严格一点应该说是硬邦邦的，可是这个声音带着一点点阴柔之气。再说了这里有一股扑鼻的清香之气，这更加不可能是易天
　　那个臭男人所能够散发的出来的。想到了这里，他也不免的睁开了双眼，尽管全身的骨头仿佛是快要被压断了一样，但是当他看到了眼前的一切的时候，他就再也不想闭上了。
　　在他的眼前是一个长发披肩，全身古朴的白色的素衣，头发上束了条金带，一张瓜子脸，双眉修长，肤色虽然雪白，但是也无法遮掩这姿形秀丽，容光照人，但是唯一的不足就是她脸色苍白，若有病容，此时的灯光很亮，可照在她脸上仍无半点血色，更显得清雅绝俗，姿容秀丽无比。世人常以“美若天仙”四字形容女子之美，但天仙究竟如何美法，谁也不知，此时一见那少女，各人心头都不自禁的涌出“美若天仙”四字来。她周身犹如笼罩着一层轻烟薄雾，似真似幻，实非尘世中人。
　　阿龙他此时恨不得自己多长几双眼睛，能够把这个美女看个够，可是让他很遗憾的是――这个美女对他的兴趣不大，她是依旧盯着他眼前的那些屏幕看着，这才让阿龙把他的注意力吸引了过来。
　　“这……这……这……这……这不就是好汉吗？”
　　阿龙他惊奇的发现在这些屏幕之中是整个小岛的监控录像，可笑他们还以为别人不知道呢？他们一直是在主人的眼皮底下干那些偷偷摸摸的事情。甚至有不少的地方还是刚刚阿龙他经过了的地方，这让身为狙击手的阿龙接受不了，要知道他们这行就是最重要的就是灵敏的感觉，要不然他们就只有死亡一途而已。可是更加让阿龙他接受不了的是――他发现他们的队长好汉傻傻的寻找着他，而四周大批的黑衣人就像是一群蚂蚁一般的冲了过去，这些黑衣人的身手非常的灵敏，受到的训练不比他们正规的特种兵少多少。这样的战斗阿龙就是用脚指头想都知道有什么下场。
