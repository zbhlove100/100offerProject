第八十章 军火之王（结局）
 　　来-看
　　“艾琳，快看看谁来了！”俄罗斯、乌克兰一趟长时间的行程过后，李韦德回到了美国国内，一进家门便嚷道。刚下飞机的时候给家里打了电话，所以他知道艾琳家，不过他想要给艾琳一个惊喜。
　　厨房内正忙碌着给李韦德做点心的艾琳，闻声跑了出来：“谁？谁来了？唉……伊夫琳！你怎么来了？”艾琳看到跟李韦德身后走进门来的，竟然是她的生死之交亲密好友伊夫琳，顿时惊喜的扑上来和伊夫琳抱了一起。
　　“克格勃解散了，的部门我没有去，不想再干那些事情了，而整个俄罗斯现又是乱糟糟的，不想继续待那里，就跟着韦德到了这里啦。”和艾琳拥抱过后，伊夫琳笑着解释，转而又轻叹了一口气，脸上摆出一副可怜兮兮的模样，“以后我可就是没有工作、没有工资的可怜人了，你和韦德可要养我。”
　　“呵呵……”被伊夫琳的表情语气逗笑的艾琳，挽着伊夫琳的胳膊边拉她进来边笑道，“你呀，这里住多久都没关系，正好我又多了一个克格勃的身保镖，这是到哪也找不到的好事哎！”
　　随后艾琳和伊夫琳笑闹做一团，又拉着伊夫琳去厨房，准备让她看看自己刚学会的手艺。不过刚进厨房，艾琳便想起了一件事，探出头来对关上大门正换鞋的李韦德说道：“对了韦德，有件事情想起来了，昨天有人寄了一封信给你，信书房桌子上摆着呢，你快去看看。”

　　“叶兰？”来到书房看到信后的李韦德，惊讶的现是叶兰的信，赶忙拆开后这么仔细的一看，“叶兰下周末也要来美国，到我这里来？太好了！”
　　这趟为期两个多月的俄罗斯、乌克兰行程，虽然也做了间人帮国方面和俄罗斯、乌克兰做了一些秘密交易，但国方面派来的一行人并没有叶兰，只是和叶兰做了一些电话交易。听说是叶兰忙一些重要的事情脱不开身，本来他还有点为她担心的呢。没想到她给了自己一个惊喜，原来叶兰竟是通过关系经过一些手续后，从总参二部退役了，要到美国来找他，这不能不让李韦德为之高兴。
　　这下家里可热闹了，三个女人一台戏，何况都是世界三大国情报机关出身，该摩擦出怎样的火花呢？李韦德有些恶趣味的想到，不过他显然忘记想了，这三个女人都来了，他该如何处理呢？
　　对了，还有李晓柔，下飞机时给她打了个电话，现也该回来了？李韦德又想到了一个女人，他的义妹李晓柔。而他刚想到这，就听到门铃想了起来。
　　李韦德赶忙去开门，虽然是豪宅，家里却并没有佣人，除了雇佣钟点工打扫一下、修剪一下草坪，因为他很少家，也不喜欢家里有个外人。
　　“晓柔！”说曹操，曹操就到，李韦德开门一看，却是李晓柔。
　　“哥！”李晓柔扑了李韦德身上，两个多月没见面了，心是十分的想念。
　　李韦德笑着把李晓柔抱怀里：“怎么按铃啊？没带钥匙么？”
　　“忘记带了。”李晓柔偷笑道，其实她带钥匙了，只是想让李韦德帮她开门而已。

　　这几年来，当李韦德把公司股票以及投资方面的事务交给李晓柔处理后，身为罗杰斯关门弟子的李晓柔完成的很出色。不但让手的那些军火公司股票、高科技公司、邮公司股票都不断的升值，还又投资或收购了一大批有着极大潜力的高科技公司，使得他们的资产是越来越多。
　　军火方面，如今美国十大军火公司：洛克希德公司、麦道公司、波音公司、诺斯罗普公司、马丁·玛丽埃塔公司、通用动力公司、格鲁曼公司、雷神公司、汤普森·拉莫·伍尔德里奇公司、联合技术公司，李韦德手都掌握了一定股份，高的洛克希德公司40%股份，少的波音公司也占了8%的股份。
　　可以说美国乃至世界军火市场上，李韦德都已经成为了呼风唤雨的角色。而国际军火市场上，那些秘密走私军火交易，李韦德也同样是屈一指的人物，别的不说，光那乌克兰3200亿军火市场70%的份额就足够傲视天下了。
　　而不仅军火市场上，石油和钻石等矿产资源上，李韦德也同样是石油大亨、矿产大亨。依靠这军火交易和附着产业，李韦德已经成为了名副其实的——军火之王。
