第二十七章 黎巴嫩真主党
　　黎巴嫩真主党是黎巴嫩什叶派的政治和军事组织，成立于1982年6月以色列入侵黎巴嫩期间，创始人是穆萨维。该组织成立初期一直处于秘密状态，1984年开始使用“真主党”的名称，1985年2月16日发表声明宣告正式成立。
　　伊朗领土与黎巴嫩并不接壤，伊朗人也不属于阿拉伯民族，但自伊朗伊斯兰革命成功以后黎巴嫩与伊朗关系非同一般。1979年成功的伊朗伊斯兰革命，推翻了巴列维王朝的统治，建立了伊朗伊斯兰共和国，确立了什叶派神职人员在伊朗的统治地位。这对于黎巴嫩什叶派是一种莫大的鼓励，一些亲伊朗的黎巴嫩什叶派激进组织随后出现，诸如“伊斯兰圣战”、“地球被迫害者组织”、“革命正义组织”等。
　　1982年以色列入侵黎巴嫩后，这些组织在伊朗的支持下逐渐联合起来，在1985年正式以“真主党”的名称亮相。真主党的宗旨正像它在1985年2月16日声明中所表达的那样：“黎巴嫩问题的解决办法是建立一个伊斯兰共和国，因为这种形式的政治体制能够确保黎巴嫩所有居民的公正和平等。”
　　至于真主党与伊朗之间的关系，上述声明中也提到：真主党“听从来自最高教法学家充满智慧和正义的领导与指示，它体现在阿亚图拉&#8226；霍梅尼这位穆斯林革命的点燃者和穆斯林复兴的引路人身上”。可见真主党毫不讳言它服从伊朗革命领袖霍梅尼的命令。霍梅尼的宗教思想和政治理念对真主党影响极大，真主党的宣传材料充满了霍梅尼的语言。
　　热衷于伊斯兰革命输出的霍梅尼对真主党十分重视，据说向真主党派去了1000至1500名伊朗革命卫队成员，帮助真主党培养骨干力量和培训战斗技能。伊朗还向真主党提供武器装备以及大笔活动经费。许多人认为，正是在伊朗的鼎力扶植下，真主党从黎巴嫩众多派别中迅速脱颖而出，成为黎巴嫩国内令人生畏的一支后起力量。
　　其实严格说起来，黎巴嫩这些势力之间的分布状况不是严格分明的，而是比较混杂的，我中有你你中有我，如犬齿般交杂着，只是在某个地区某个势力占据强势而已。
　　但不管怎样，越混乱越复杂的局面状况，对李韦德来说是越有利。
　　针对这么多的势力和武装组织，李韦德在掌握了基本信息后，也进行了分析选择。
　　首先什叶派阿迈勒民兵和德鲁兹民兵，李韦德是不会选择的，因为他们背后有叙利亚，以及更深的苏联，他们并不缺军火支援。其次长枪党也不是好的选择，因为他们背后是美国为首的西方国家，并且还有军舰的强力支持。逊尼派武装不用考虑，一个自是自卫的温和派武装，实在不对军火商的胃口。
　　那么剩下的目标，就只能是重返黎巴嫩南部地区急需恢复元气的巴勒斯坦解放组织武装力量，以及需要壮大自己实力的真主党武装游击队了。这些也正是李韦德所选择的目标。
　　临近中午，在黎巴嫩首都贝鲁特市南郊的什叶派穆斯林居民区，某处看似非常普通的居房内，正有面对面的两人用阿拉伯语聊着事情。
　　“真主在上！美丽的幽兰，离我们上次在石家庄步兵学校见面，匆匆一别已有一年多了吧？没想到贵国派出的会是你，真是让我深感荣幸！”正在微笑着说话的，是一个留着胡子、穿着深色袍子、戴黑色头巾，一副穆斯林教徒打扮的二十五岁左右的黎巴嫩年青人。
　　而他对面坐着被称为“幽兰”的，则是一位传统穆斯林妇女打扮的女性，因为被仅仅只在眼前留一条缝隙的黑色面纱挡住，完全看不到面貌以及任何表情，只能隐约看到秋水般的黑色眼眸，以及听到一个十分悦耳的笑声传出：“哈桑，时间过的也并不太久嘛！不过上面之所以派我来中东，主要倒不是为了让我监送去年答应你们的那批军火来黎巴嫩的，而是为了别的国家另外一件重要的事情，来你这里可只是顺路哦！”
　　名为哈桑的黎巴嫩年青人，听了幽兰的打趣并没有丝毫的介意，而是哈哈大笑道：“你能来就是我们的荣幸了，可不敢让你专程为我们来跑一趟。嗯……叶元帅他老人家还好么？”
　　“爷爷目前身体还算可以，只是上个月得了感冒，修养了几天就好了。”幽兰说道。
　　“愿真主保佑他老人家！”哈桑做了个祈祷祝福一下，然后惋惜的说道，“可惜我三次去贵国，却因为种种原因不能见到他老人家。像他老人家那样和毛主席同一时代的军事大家，不能够见一面真是太遗憾了！”
　　“自从多年前一次偶然的机会，我开始研习中国战争理论，就被中国的兵法艺术深深震撼了。特别是其中的游击战、地道战、人民战争，是最适合我们这样的武装组织了的。我在1982到1985年向组织提议，派出四十多名组织中高层骨干，以伊朗军事人员的身分前往中国，在石家庄步兵学校留学三年多，也正是因为如此。可惜我因为组织里众多事情要忙，只能去那短暂旁听三次，不然我真想在那里系统的多学几年。中国真是一个伟大的国家！有如此灿烂历史和文明，如此高深的智慧和军事艺术，相信你们一定能够彻底的强盛起来！”
　　“谢谢你的吉言！”幽兰笑着答谢，“对了，你们组织还会派遣骨干去我们中国石家庄步兵学校留学么？”
　　“这个说不定，目前是没有计划。那四十多名留学归来的骨干，已经被分派到我们游击队各处担当军事教官和指挥员了，足够支撑起现在整个组织的武装指挥力量。目前我们需要的是军火，基层士兵和民兵的武装训练，以及进行实际的行动。当然，如果我组织进一步的壮大到某种程度，我想我们还是会再次派遣骨干去贵国留学的。”哈桑笑着说道。
　　“祝愿你们能够成功，实现你们伟大的理想！”幽兰现在也回报对方以祝福，在哈桑同样答谢后，转而又说道，“不过现在，我们还是回到正题谈谈吧！”
　　哈桑笑了笑正要回答，这时却忽然响起了敲门声。
　　“进来！”哈桑皱了皱眉头，在看到自己的手下进来行礼后，略显不快的轻声喝道，“之前我不是说过，不要来打扰我们的谈话了么？现在怎么还来？到底是发生了什么事情？”
　　只见那名手下再次鞠躬行礼后说道：“尊敬的纳斯鲁拉，是有三个自称是苏联军火商的人前来拜访，而他们的介绍人是哈米尔，手中拿着哈米尔的亲笔信，所以我才来请示您的。”
　　“苏联军火商？哈米尔？”哈桑口中喃喃的重复着这两个关键词，没有了被打扰谈话的不快。
　　哈米尔是谁？苏联克格勃在黎巴嫩的情报负责人（表面公开的，而且只是代号）。黎巴嫩不管是哪种势力，多多少少都要给点面子，即使是以色列或者美国支持的基督教派和长枪党，也不会正面与之发生冲突。因为黎巴嫩的局面情况已经够乱够复杂的了，谁都不想无事制造借口让苏联进来参合一脚甚至与之为敌，即使在黎巴嫩最大势力叙利亚的背后有着苏联这个盟国，但那毕竟是间接的，苏联一直没怎么对黎巴嫩感兴趣过。
　　现在哈米尔介绍三个人来，对方自称是苏联军火商，看来是真的了。苏联自己的组织惹不起，何况自己组织背后最大的靠山是伊朗，伊朗又是苏联的盟国呢。而军火商么，现在军火可是自己组织所需要的东西，送上来的买卖自然不会拒绝。
　　“嗯……幽兰，你看呢？”尽管哈桑初步有了决定，但出于礼貌以及其它缘故，还是向坐在对面的幽兰询问了下看法。
　　“当然要见见！如此重要而有趣的人物来访，怎么能闭而不见呢？哈桑，你不必介意我在。”幽兰毫不犹豫的说道。碰到这样的事情她怎能不看看听听？至于她自己么，可是穿着那极度保守的传统伊斯兰妇女服装，黑色的全覆盖面纱让她的面貌没有一点的泄漏，自然更不会让不该知道的外人知道自己的真实身份。
　　看到幽兰也同意，哈桑便对那名手下说道：“你去让他们进来吧！嗯……是三个人吧，你让为首的人进来就行了，其他两个你让他们在外屋等候，当然一定要礼貌的招待他们。”
　　“是！”那名手下听到吩咐后，便行礼退出去了。
　　过了一小会儿，就见那名手下领着一位满头耀眼的金发、精美雕刻般的英俊脸庞、身高一百八十五公分以上、体格壮实健美匀称的人物走了进来，这个人正是来到黎巴嫩做军火生意的李韦德。
　　只见李韦德进来后打量了屋内的两人，然后对哈桑微微致意，用阿拉伯语微笑着说道：“哈桑-纳斯鲁拉先生，很高兴见到您！顺着真主的指引，我来到了这里，为贵组织为真主党带来了军火！”
　　李韦德根据从伊夫琳那边得来的情报，很清楚眼前这位黎巴嫩年青人，在真主党内的影响力。
　　虽然这个名叫哈桑-纳斯鲁拉的年青人现在才26岁，在真主党里又没有担当具体的高级职务，也并不是真主党最高领导机构――由12人组成的协议委员会的委员。但他却是真主党的领袖赛伊德-阿巴斯-穆萨维总书记最得意的学生、心腹助手兼秘书，并协助老师穆萨维参与了真主党的组建工作。
　　哈桑-纳斯鲁拉1960年出生于黎巴嫩首都贝鲁特郊区，一个贫穷的什叶派穆斯林家庭。16岁那年纳斯鲁拉到伊拉克南部城市纳杰夫求学，找到了当时在纳杰夫神学院负责新生的教士赛伊德-阿巴斯-穆萨维。在恩师的悉心点拨和传授下，纳斯鲁拉潜心于宗教学业，他只用了两年时间，就学完了其他人需要5年才能学完的课程。
　　1978年纳斯鲁拉回到黎巴嫩，次年加入了阿迈勒民兵，担任贝卡地区的领导人。1982年以色列军队入侵黎巴嫩，近60万名什叶派穆斯林难民涌入贝鲁特南郊。这些衣食无着的难民希望拥有自己的组织，于是纳斯鲁拉脱离阿迈勒民兵，在伊朗精神领袖霍梅尼的支持下，和老师穆萨维一起创建了真主党。穆萨维被推举为首任总书记。
　　正是因为这样，李韦德作为一个初次见面的军火商，虽然有苏联克格勃的介绍，但也不可能直接去设在贝卡谷地的巴勒贝克市的真主党总部，见到真主党领袖穆萨维的情况下，只有先来贝鲁特市南郊见见这个纳斯鲁拉了，只有从他这里打通生意的渠道。

　　PS：石家庄步兵学校，1992年8月改为陆军参谋学院，1999年改名为石家庄陆军指挥学院。
　　真主党几十名骨干1982到1985年以伊朗军事人员的身分，来我们中国石家庄步兵学校学习了三年多，这可绝对不是我恶搞，而是真实的事情！
　　中国在中东的影响力绝对不可小瞧，这也是为什么中东一出什么比较大的事情，中国外交部就要面对媒体发表一下看法，大家可别觉得这是无聊，也不要以为仅仅因为中国是联合国的五个常任理事国。
　　另外有很多人问过一个问题，所以我在此说明一下，李韦德既然以后被称为军火之王，自然他的生意范围不仅仅是地下军火市场，他今后也会以合法的军火商身份出现，比如洛马公司那样国家支持的合法军火生产商和销售商，参与国家对国家的军火贸易。
　　黑道和白道通吃，才是王道！
