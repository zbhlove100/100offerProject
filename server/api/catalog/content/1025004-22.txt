第七节 雨林大混战
　　阿龙他在专心致志的狙击敌人的狙击手的时候用眼角看到了一个诡秘的黑影朝着他扑了过来，他就在这生死立判的情况下，他掏出了随身携带的手枪，可是他尽管很快，不过那个黑影的速度或者说是在他的身手在阿龙他目前的这样状态下是差了一点，他就是感觉到他握枪的那只手有一股大力传了过来，使得不得不把他的枪口掉转方向，而且不但如此他还看到那个黑影是离他越来越近了，在那黑影的上方有一个发亮的东西，这个不明的发亮的东西它以很快的速度朝着阿龙压了下来。
　　原来是那个黑影在看到那阿龙掏枪的时候，他的脚微微的以蹬，立马是加快了速度，他几乎是撞了上去，他的手用力的岔开了阿龙握枪的手，同时他另外拿匕首的手以更快的速度准备狠狠的插向阿龙。
　　几乎在这个生死关头，阿龙他发挥出来了他作为一名合格的特种兵的素质出来，他握枪的手立刻是放弃了争斗，因为他知道那样是没有用的，于是他只能是用另外一个更加好的方法来解决这个问题了。他的脚迅速的抬了起来，踹在了来袭黑影的肚子上，并且以这个为支点把他狠狠的掀翻到另外一边去了，不过估计那边的小石块应该不少，让那个家伙吃了不少的亏，因为阿龙他听到了一阵闷哼的声音。
　　阿龙他现在没有立刻反击这个黑影，因为他现在有更加重要的事情要做――警告那些仍然是处于不知情的战友和在他眼前端着自动步枪冲了过来的一个武装人员。阿龙能够原本是放弃了抵抗的握枪的手迅速的抬了起来，平躺着身体对着那跑过来而且很有可能马上就要射击的武装人员就是几枪。
　　――嗖嗖嗖――
　　几颗快速飞行的子弹速度击中了他的脑袋，阿龙他就是听到了一个像是西瓜被击碎掉了的声音，接着是一个没有脑袋或者算是半个脑袋的尸体无力的带在惯性跌到地面上。阿龙他虽然是解决掉了一个非常危险的敌人后，但是他还是还有很多的危险没有解决，比如首先就是那个被掀翻掉了的黑影，那些可能存在的和这个黑影以及那个被阿龙打死掉的武装人员的同谋，还有一些没有解决的伏击他们的武装人员，同时大批不知道还在哪里的这个幽灵岛的秘密部队。
　　远处处于激烈的激战的好汉他是听到了从阿龙那个方向传来了一阵让人揪心的枪声，这让他头晕不已，因为他不知道是不是今天出门没有看皇历的缘故，反正他现在是不知道为什么会一波为平而另外一波又起来了。他低声的对着身边的无天说道：“无天！好象阿龙那边有了什么麻烦了？”
　　无天他也非常着急的说道：“是啊！我现在也是非常的着急，因为我们真正的目的是要锻炼阿龙的，而不是……”
　　好汉立刻阻止了无天他要说出来的话，好象他们来这里完成这么一个超级艰难的任务而不是为了国家安全而是为了一个乳臭未干的小子。好汉他说道：“不要说这些废话了，如果阿龙他真的有什么三长两短的话，我们怎么对得起教授呢？”
　　于是在他们的的示意之下，他们纷纷的后撤了，当然了这些情况全部都是发生在对方的眼皮底下，但是身为在死亡边缘打滚的人，他也知道对方是发生了什么情况，他立刻指挥着他那剩下不多的人，要求他们立刻攻上去，解决掉对头。
　　一旁的副队长连忙说道：“队长！我们是低估了这些中国人的实力，现在我们是在打伏击的情况下还死了这么多人，如果是在平地上遭遇那还不是要全军覆没了吗？”
　　那个让人发寒的家伙，冷冷的一笑，他说道：“这么说来你是认为我是错了，而你是对的了吗？”
　　那个副队长好象是感觉到了什么一样的，他用颤抖的声音说道：“没……没……没……有！你一……直是……我们的……灯台！我从……来没有其……他的意……见！”
　　――砰――
　　冷血的队长吹了吹枪口，尽管因为现在的枪口并不会因为发射了子弹而冒出青烟，不过不知道从什么时候开始，他就喜欢上了这个习惯――每当他杀死了一个同伴都会这个样子。
　　他冷冷的说道：“你对我没有意见，但是我对你有很大的意见！再见了！我的第12任副队长！”
　　在另外一个神秘的地下室里面，这个地方是非常的乱，因为这里是有很多的人在这里走来走去的，而且还有很多的通讯器材以及一个超大的液晶屏幕，这一切就看上去像一个临时的作战指挥中心，不过这样的一个说法其实还是挺正确的，因为这里才是或者说是才算是一个真正的作战指挥平台――一个守卫在这里的佣兵团。
　　一个壮汉满脸怒意的看着一张纸，这张纸上面写的是他们刚刚遭受到的损失，他们虽然是世界上最出名的佣兵团，他们手底下的佣兵也是非常的精锐，可是他没有想到遇到了一个这么难缠的对手。
　　――梆――
　　他愤怒的拍打了一下桌子，他大声的怒叫道：“到底是什么人？是那些该死的中国人吗？我一定要拔了他们的皮，害的我们佣兵团几乎是名存实亡了！”
　　站在壮汉周围的几个人看到这个准备要把人撕成碎片的壮汉纷纷变成了一群看到了老虎的绵羊一样，动都不敢动了。就在此时一个人冲了进来大声的说道：“老大！我们发现他们了，他们在X大桥附近，根据我们的人所说，他们在那个方向听到了激烈的交火的声音，估计是我们的人发现了他们。”
　　这个壮汉就好象是吃了兴奋剂一样的兴奋起来了，他大力的拍打着桌面，那惊人的力量几乎把这个桌子拍碎了。他大声的吼叫道：“那还等什么？难道要等我请你们吃消夜吗？赶快出动！把他们抓回来，我一定要把他们撕成碎片，让他们知道我们烈炎佣兵团的厉害。”
　　如果是其他的对世界的地下佣兵团的势力有了解的人听到了烈炎佣兵团这几个字一定会大吃一惊甚至是立刻打起抖来，因为这是地下十大佣兵团之一，尽管这几年不知道为何十大佣兵团里面的后面七支神秘的消失了，而使得后面的七支佣兵团有机会晋升成为世界地下十大佣兵团，但是他们后面的七支的实力也是同样的非常出色，而其中以烈炎佣兵团的脾气最为暴烈，他们对敌人一向是冷酷无情而闻名。
　　阿龙他在解决掉了一个他认为的一个比较危险的麻烦后，就开始准备着手对付另外一个麻烦了。那个黑影因为把阿龙狠狠的掀翻到了到处都是小石块的地面上而让他的后背着了大罪了。但是他毕竟是究竟训练的人，他立刻一个标准的鲤鱼大挺，尽管不知道他是什么国籍的，到底会不会这么一个高难度的动作。可是他毕竟是站了起来，他用充满了愤怒的眼神盯着远处的阿龙，特别是他看到了阿龙射杀了他的一名同伴的时候，他再一次暴跳起来了，手再一次的紧紧的抓住了他那锋利无比的匕首，而为了能够让他的匕首更加具有攻击力，他还特地的把这只手稍微的向后面缩了缩。
　　阿龙他准备转过身体对付那个比较难缠的家伙，尽管他知道他的背后的草丛里面可能还有一个瞄准他的人，不过如果他视他后面这个拿匕首的黑衣人如无物的话，估计他就会立刻受到很重的伤。阿龙他其实是非常的讨厌做这样的抉择，因为这样的选择题你不能靠自己，只能是希望对方可以放过自己，这对于终日要在生死边缘混的阿龙来说是一个难以选择的或者是没有经历过的。
　　不过两害取其轻还是听说过了的，而且对于阿龙来说也并不是没有一丝一毫的机会，他没有用手枪，因为在身体肉搏的过程之中，这个玩意还不如用匕首来的稳当。于是他也是抽出了一直随身携带的虎牙。
　　阿龙是正手握住匕首，那匕首是笔直的朝着奔来的黑衣人插了过去，以虎牙的锋利程度，阿龙他丝毫不用担心不能切开这个脑袋。黑影他的眼睛微微一迷，原本缩在后面的匕首此时就像是安装上了火箭的发动机一样以令人难以置信的速度划了过来，如果以这样的速度不变的话，黑影的匕首应该可以准备无误的切断阿龙的手腕。
　　――铛――
　　一阵金属撞击在一起的声音，原来就是刚才的一瞬间，阿龙他那灵活的手腕突然变化起来了，匕首一转，原本是正手握住匕首的姿势变成了反手握匕首，还这个黑影的姿势是一样的。于是使得原本应该切断阿龙手腕的匕首和阿龙他的虎牙撞击到可一起。
　　而强大的撞击力把阿龙握住匕首的手撞开了，阿龙就借着这样的一个力，凌空回旋的踢腿，准备给黑影的脑袋上来这么一下。但是黑影他是微微的一蹲躲了过去，接着趁阿龙他还没有站稳脚跟的功夫，连忙是冲了过来，各种各样的搏击术一起施展开来了。阿龙他也是用了吃奶的力气挡住了黑影的进攻，不过他已经是足足后退了十多步。
　　阿龙他马上摆毫了防御的姿势，慢慢的放缓了他的气息，平静了一下他此时的心情，而另外一边的黑影也是做出了同样的动作。看样子他们都是打上了瘾，毕竟是要找一个和自己不相上下的对手很不简单。
　　也就在这样的短暂的瞬间阿龙他又后悔了，因为他隐隐约约的听到了一阵急促的脚步声朝着他的后面跑了过来，而他知道从那个发现来的人可能会是各种各样的人，但是绝对不会是他的战友，于是他就打定了主意准备开溜，反正他是警告了好汉他们，而以好汉他们老道的经验来看，没有道理还吃他们的大亏。不过阿龙他是没有发现这个黑影也打了同样的主意，因为好汉他们朝着这个方向撤退了。不过他的脸上没有露出任何想逃跑的神情，就是露了出来也没有多少人能够发现，因为他的脸蛋上尽是那些迷彩，上面有上面表情也看不出来。而阿龙也借着这个双方对峙的时间好好的观察黑影，因为在刚才那一眨眼的功夫他们进行了激烈的打斗，是阿龙他出道以来最为难缠的对手，阿龙回想起刚才他们的交手再观察这个较为弱小的身躯，觉得这个人应该走的是灵活路线，不过这也有一点不对，以阿龙他的身体状态在那些特种部队里面走的也是灵活路线，而在他面前的这个更为弱小的身躯，让阿龙觉得有一丝异样。
　　突然间，对面的黑影他动了，他首先的打破了这样的尴尬的状态――既双方不敢乱动怕遭到对手的反击而同时也背后有很大的危机。阿龙他算是松了一口气，不过他没有打算放过这个黑衣人，他冲了上去，再次改反手为正手挥舞着匕首刺了过去。那个灵巧的黑衣人他那矫捷的步伐远在阿龙他之上，他轻而易举的避开了阿龙他的突刺。阿龙他看到他的招数没有击败对手，连忙后撤因为他已经是听到了他的脑后有一阵风声吹了过来，他连忙把他的另外一只手握紧，狠狠的砸了过去，和这个黑衣人的脚撞击到了一起，他们后退了好几步，这样以来他们回到了刚刚开始的时候的位置，而且不但是摆脱了背后的危机还和自己的队伍站到了一起，可谓是皆大欢喜。
　　阿龙他此时突然想了起来他那刚刚被迫扔掉的手枪，于是他一个翻身滚到了离他不远的手枪的旁边，一把抓了起来，同时朝着脑海中预定好了的位置射击。不过那些子弹只是把那些枯枝败叶打的更加的碎而已，因为阿龙他的动作虽然是很块，可是那个黑衣人他的反应有不慢，当他看到阿龙好好的打起滚来的时候，他就下意识的感觉不对劲了，他是早一步的跳了开来，因此他就避免了致命的一击。
　　阿龙他看没有命中黑衣人后，准备接着射击，可是有一阵很打的力量朝着他扑了过来，让他的重心变的不稳和那个来人滚到了一起，而他的脑门上有几梭子弹飞了过去。阿龙他回过头一看，原本不是别人正是他们最为机敏的夹子。夹子看到阿龙呆呆的看着他，他不由的笑了笑，说道：“你啊！这样抬危险了，你要知道在战场上什么让都有可能出现在你的背后，以后你一定要小心才好哇！”
　　阿龙他充满感激的点点头说道：“我会的！谢谢你！”
　　夹子大力的拍了拍阿龙的身体说道：“我们之间不要谈这个字，不过你要记住了你歉我一条命，以后要记着还！”
　　好汉他是非常的郁闷，因为他看到那些该死的跟屁虫居然还是拼命的紧跟着他们，而前面的那些来历不明的武装人员也让人挺烦的。想到了这里他不由的叹了一口气，他问了问阿龙。
　　“阿龙！你怎么不早点说你看过这些人呢？现在让我们很被动，知道吗？”
　　阿龙的低着头，低声的说道：“对不起！我也是在解决掉了那些特勤处的人之后，隐隐约约的看到了他们，而我们完成任务比较顺利，他们也没有立刻找我们的麻烦，所以我也一时忘记说了！”
　　无天他对阿龙说道：“阿龙！你以后要把战场中发生的任何一处细节都要想清楚才行，应该你任何一丝疏忽掉了的细节都有可能造成我们明日死亡的原因。所以我们有一句老话，绝对不放过任何的一丝危险信号！”
　　目前他们三方势力处于非常微妙的状态，其中以阿龙他们的实力最强，尽管他们的人数可能不是最多，但是他们却是这里素质最好的一群人，而他们在被伏击之中还拿重创对手就是一个鲜活的例子。其次就是黑衣人他们一伙，因为他们还没有发生什么大的交火事件，所以他们目前的战斗力对为强悍，最后就是那些倒霉的伏击者，他们不但没有捞到什么好处反而是遭受到了重创，本来他们还不够那些人看，不过在这样一个比较微妙的关头他反而是起到关键作用的平衡器，无论他是压到那一边都拿起到很大的作用。阿龙他是低着头躲到了一个比较隐蔽的地方，因为身为狙击手的他们无疑是第一波的攻击目标，他也是非常晕的看着这样的局势，三方人马在茂盛的热带雨林之中静静的对峙着，时不时的有几只不长眼的小动物在大批的人的眼睛的可视范围之内不停的蹦来蹦取，而阿龙他还算比较的幸运，只不过有一条蛇在他的旁边，不断的吐着它的舌头，就好象好热一样。通过瞄准器，阿龙看到一个倒霉鬼被蚂蝗给咬住了，大量的鲜血源源不断的流进了这个吸血鬼的肚子里面，那细小的蚂蝗也因此暴涨了数倍。
